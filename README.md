# Interactive Jet Parallax Website ✈️

An immersive, interactive parallax scrolling experience that takes visitors on a journey through a commercial aircraft!

## 🎯 Features

### Visual Effects
- **Parallax Scrolling** - Layers move at different speeds as you scroll
- **Smooth Animations** - Floating jet, spinning turbines, pulsing indicators  
- **Interactive Elements** - Clickable seats, cabin door, and more
- **Responsive Design** - Works on desktop, tablet, and mobile

### Journey Through the Jet
1. **Hero** - Meet the jet with floating animation
2. **Engines** - Spinning turbine visualization
3. **Cockpit** - Flight deck with animated gauges
4. **Cabin Entry** - Interactive cabin door
5. **Main Cabin** - Seating configuration (click seats!)
6. **Entertainment** - In-flight screens
7. **Galley** - Catering area
8. **Lavatory** - Facilities
9. **Safety** - Emergency exits
10. **Cargo** - Storage hold
11. **Landing** - Final descent & thank you

## 🎮 Interactive Features
✨ **Click the jet 3 times** to make it fly away (easter egg!)
🖱️ **Click seats** to select them
🚪 **Click the cabin door** to open/close
⌨️ **Arrow keys** to navigate between sections
📊 **Scroll progress bar** at the top

## 🛠️ Customize It

### Change Colors
Edit gradients in `styles.css`:
```css
background: linear-gradient(135deg, #your-color-1 0%, #your-color-2 100%);
```

### Adjust Animation Speed
Modify durations in `styles.css`:
```css
animation: float 3s ease-in-out infinite; /* Change 3s */
```

### Change Parallax Speed
Adjust data-speed in `index.html`:
```html
<div class="parallax-layer" data-speed="0.5"> <!-- Smaller = slower -->
```
